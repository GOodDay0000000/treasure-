### 주요 결정사항
- 번역본 데이터: assets/bible/{version}/{bookKey}/{chapter}.json
- 찬송가: GitHub Release 다운로드 → 앱 내부 저장
- 다국어: 하드코딩 맵 방식 (15개 언어)
- 다크모드: SharedPreferences 저장
- 클래식 모드: SharedPreferences 저장
- 연구모드: 완전 삭제
- 주석 버튼: AppBar 배치 (데이터 준비 후 연결 예정)
